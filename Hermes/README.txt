It is important to know if you want to explore this project to see how it looks or to experiment with it then do the following.

1. Open XAMPP
2. Turn on Apache and MySQL
3. ONLY COPY the folder Hermes to the HTDOCS folder in the xampp directory
4. there is an sql dump of the database called hermes.sql
	filepath   /hermes/hermes.sql
	copy the contents and run the SQL code in localhost/phpmyadmin to create database environment
5. do the above steps to make sure hermes runs correctly.
