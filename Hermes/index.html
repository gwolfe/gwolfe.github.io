Information

This Project is designed to be a web based communications tool for parents and teachers. To enable the parents and teachers ease of communication without having to spend time sending countless emails and calls without getting an instant reply. Using a web based application allows for information anywhere, any time and at the convenience of the end users. Hermes is not software to be installed but an accessible web application for any mobile device, whether that is a laptop, tablet or even a mobile phone. It is perfect for any busy teacher or parent that is on the go non- stop.
This Project was initially thought of for teachers and designed with their needs in mind with some aspects designed for the parents. The requirements for this project will be derived from feedback teachers gave during a survey in the hope of creating a potentially usable web application. Because parents and teachers can range from any age and are not always tech-savvy the design and layout of the web application will be very simplistic with very little navigation needed and it will have a secure login system so no private information can be stolen.

Functionality:


Calendar,
Creating/Deleting a Class,
logging in/Registering/Logout,
Editing Account,
Messaging.
